# Snaik
A simple web-based snake game built using HTML, CSS, and JavaScript.
## How to Play
1. Use the arrow keys or WASD to move your snake
2. Use SPACEBAR to Start/Restart the game
3. Use ESC to Pause the game
## How to Run the Game Locally
No installation is required! 
1. Download or clone this repository.
2. Open the `index.html` file in any web browser (Chrome, Edge, Safari, Firefox).
